# Honors-Programming-Principles-Project- Task Tracker Program

## Description
This Java program helps to keep track of tasks and dealines. Users can add tasks, view them, mark them as completed, remove tasks, and see a summary of their progress. I created this project to build something practical while applying Programming Principles 1 concepts. 

## How to Run
1. Open up the project in your Java IDE>
2. Run the Man.java file.
3. Follow the menu options in the console

## Features
-Add new tasks
-View all current tasks
-Mark tasks as completed
-Remove tasks
-View a progress summary
-Handles invalid input with exception handling

## Programming Concepts Used
-Variables and data types
-Conditional Statements
-Loops
-Methods
-ArrayLists
-Classes and objects
-Exception Handling
-String Formatting

## Challenges I faced
One of the biggest challeneges I had was making sure all three classes worked together correctly. I also had to fix issues with input validation and adding the remove task feature. This project helped me get more comfortable fixing errors and organizing code across multiple classes. 

## Future Improvements
I would add file saving so tasks could still be there after the program closes. I would also try to improve user interface to make the program as easy to use as possible.
